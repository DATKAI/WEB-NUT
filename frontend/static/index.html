Вариант 2 — Служба Windows (без входа пользователя)

Устанавливает PowerShell мониторинг как настоящую службу Windows через NSSM. Запускается автоматически при старте ПК от имени SYSTEM — без входа пользователя. При низком заряде выключает ПК.

⬇ Скачать nut-service-install.zip
1
Распакуй архив и запусти nut-service-install.bat от администратора

Правой кнопкой → «Запуск от имени администратора». Скрипт скачает NSSM и установит службу автоматически.

2
Что установится

Служба NUT-Monitor в services.msc — тип запуска «Авто», работает от SYSTEM. Лог: C:\NUT-Monitor\nut-monitor.log

3
Удаление
powershell -File nut-service-install.ps1 -Uninstall

ntfy — Push уведомления без VPN

Бесплатный сервис push-уведомлений. Работает в России без VPN. Уведомления приходят на телефон мгновенно.

🚀 Быстрая настройка (2 минуты):

1. Установи приложение ntfy: Android / iOS
2. Придумай уникальное название темы, например: nut-monitor-иванов2024
3. В приложении нажми + и подпишись на свою тему
4. Введи URL ниже и нажми Сохранить

Формат: http://ntfy.sh/название-темы (используй http, не https — HTTPS может зависать)
